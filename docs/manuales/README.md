# Manuales Internos CRM Cuid-Arte

Este espacio concentra la documentacion interna operativa para coaches y perfiles de gestion.

## Documentos disponibles

- [MANUAL_COACH_PORTAL.md](./MANUAL_COACH_PORTAL.md): guia completa del flujo del coach en CRM y Portal Cliente.
- [SOP_COACH_SEMANAL.md](./SOP_COACH_SEMANAL.md): protocolo estandar diario/semanal/mensual.
- [FAQ_COACH_Y_SOPORTE.md](./FAQ_COACH_Y_SOPORTE.md): incidencias frecuentes y resolucion rapida.

## Uso dentro del CRM

- Vista interna: `Manual Interno Coach`.
- Ubicacion menu: `Seguimiento de Clientes > Manual Interno Coach`.
- Roles con acceso: `coach`, `admin`.

## Como actualizar estos manuales

1. Editar el archivo correspondiente en `docs/manuales/`.
2. Añadir al final del documento una linea en `Historial de cambios`.
3. Hacer commit con mensaje claro (ej: `docs: actualizar SOP semanal de coaches`).
4. Notificar el cambio en `Tablon de Anuncios` para que todo el equipo lo vea.

## Convenciones de contenido

- Escribir en lenguaje operativo (accionable, sin ambiguedad).
- Priorizar formato `Si pasa X -> hacer Y`.
- Mantener checklists cortos y ordenados por prioridad.
- Evitar duplicar criterios: si una regla ya existe en otro documento, enlazarla.

## Responsables sugeridos

- Dueno funcional: Head Coach.
- Dueno de calidad operativa: Direccion.
- Dueno tecnico (estructura y navegacion interna): equipo CRM.
