## 6. DURACION Y CANCELACION

### 6.1 Duracion
Este contrato tiene una duracion **indefinida** desde la fecha de firma, renovandose automaticamente cada mes.

### 6.2 Cancelacion
Cualquiera de las partes puede cancelar el contrato con un **preaviso minimo de 30 dias**.

### 6.3 Proceso de baja
Tras la cancelacion:
1. EL PROVEEDOR exportara todos los datos en formato estandar (CSV, JSON)
2. EL CLIENTE dispondra de 30 dias para descargar sus datos
3. Transcurridos 30 dias, los datos seran eliminados permanentemente
4. Opcionalmente, EL CLIENTE puede solicitar el codigo fuente para auto-alojamiento

### 6.4 Reembolsos
No se realizaran reembolsos por periodos parciales no utilizados.
